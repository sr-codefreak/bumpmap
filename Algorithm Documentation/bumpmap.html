Placing the Bump

The bump created at O(0,0,0) by changing the normals of various points to get the bump. It is then translated to Q(0,0,1). Next we consider the vector to any point P(x,y,z) on the sphere. We calculate the cross product of OQ and OP to get the normal vector N of the plane formed by the points O, P and Q. Then we calculate the angle between OQ and OP in the plane using the formula for dot product of OQ and OP. Next we do rotation transformation by that angle about the N axis. Thus the bump is now placed and correctly aligned for the point P(x,y,z). We do this for some more points to get some kind of a symmetry or randomly.

Random Bumps

The genpos() function randomly generates approriate points on the sphere and also the size and shape of the bump. Using these values in the pos[][] array these bumps are generated and scaled randomly in the x and y direction to make them elliptical.
